# StockWatch

# Overview
> Kotlin-based Android app helps users to tracking the US Market Stock.


# Tools and Tech stack used :

• Kotlin, Hilt, Fragments, Retrofit

• OpenCSV

• Coil

• Custom Chart to showing the stock movements

• Room database
